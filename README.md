# MEPFacilities Corporate Website

## About the Company

**MEPFacilities** is a Spanish business group specialized in MEP (Mechanical, Electrical, and Plumbing) engineering and comprehensive project execution. The group integrates three specialized companies:

### Companies in the Group

1. **GS Fontanería** - Plumbing specialists
   - Water supply installations
   - Sanitation and drainage systems
   - Residential, commercial, and industrial projects

2. **Jamafrio** - HVAC & Climate Control
   - Industrial and commercial HVAC systems
   - Industrial refrigeration
   - High-performance climate control solutions

3. **MEP Projects** - Engineering & Project Management
   - MEP project design and planning
   - Mechanical, electrical, and plumbing engineering
   - End-to-end project execution

## Technology Stack

- HTML5
- CSS3 (Vanilla CSS with CSS Variables)
- Vanilla JavaScript
- Google Fonts (Inter, Poppins)
- FormSubmit.co for contact form
- IntersectionObserver for scroll animations

## Key Features

- Dark theme with green accents
- Professional scroll-reveal animations
- Responsive design (desktop + mobile)
- Functional contact form with email sending
- Premium visual design with glassmorphism effects
- Floating logo cards in hero section
- Stats counter animation
- Card hover effects

## Running Locally

Simply open `index.html` in a browser. No build process required.
